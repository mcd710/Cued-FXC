Instructions-COLLECTOR

When you see these two symbols at the start of your turn, you will play the role of COLLECTOR. The symbols also tell you the value of each colored word on that turn. At the end of each turn you will see how many gems you created on this turn.

Remember, this image means that you are on a LOW-VALUE turn, where each correct response will create only gem.

Remember, this image means that you are on a HIGH-VALUE turn, where each correct response will create gems.

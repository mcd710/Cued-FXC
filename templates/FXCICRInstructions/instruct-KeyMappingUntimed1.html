Instructions

In this game you will see words printed in colored ink on the screen.

Before we begin the game we are going to have you learn which color ink corresponds with each key.

You will see a row of Xs appear on the screen in different INK COLORS.

For each ink color you should press the key that corresponds to that color ink.

This practice is untimed so you have as long as you would like to respond.

After each response, you will be told whether you were correct or incorrect.

Please do your best to make sure you learn which key corresponds to which ink color.
